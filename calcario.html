Getting Started

Calcario is a browser-based mathematical computing environment that supports both numerical and symbolic calculations — no installation required. It is powered by Math.js for numeric/matrix work and Algebrite (a JavaScript CAS) for symbolic algebra and calculus.

The Interface

  • Input bar (bottom) — type any expression and press Enter to evaluate.
  • Output area — shows each command and its result. Results are color-coded: black for numbers, green for symbolic expressions, red for errors.
  • Workspace panel (right) — lists all variables currently defined in the session.

Two Computation Modes

Calcario automatically chooses the right engine based on context:

  • Numeric mode (default) — evaluates expressions to numbers using Math.js. All standard arithmetic, functions, and matrix operations run here.
  • Symbolic mode — activated when symbolic variables are declared with syms, or when symbolic functions like diff, integrate, factor are used. Results are kept as exact algebraic expressions.

Quick Start Examples

>> 1> 2 + 3 * 4              % arithmetic
ans = 14

>> 2> sqrt(2)
ans = 1.41421

>> 3> A = [1, 2; 3, 4]        % 2×2 matrix
A =
   1   2
   3   4

>> 4> syms x                  % declare symbolic variable
>> 5> diff(x^3 + 2*x, x)
ans = 3*x^2+2

Suppressing Output

End any command with a semicolon ; to suppress its output (the calculation still runs and the variable is stored):

>> x = 1000;        % silent — no output printed
>> y = x / 4        % output shown: y = 250

Command History

Press ↑ / ↓ arrow keys to scroll through previous commands. Press Ctrl+L to clear the terminal.

Comments

Anything after % on a line is a comment and is ignored by the engine:

>> det(A)    % computes determinant of A
💡 Variables persist for the entire browser session. Use clear all to reset the workspace.

Matrix & Linear Algebra

Matrices are first-class objects. Use square-bracket notation with commas separating elements and semicolons separating rows — identical to MATLAB/Octave syntax.

Creating Matrices & Vectors

>> v = [1, 2, 3, 4]          % row vector (1×4)
>> A = [1, 2; 3, 4]           % 2×2 matrix
>> B = [1,0,0; 0,1,0; 0,0,1]  % 3×3 identity (manually)
>> r = linspace(0, 1, 5)      % [0, 0.25, 0.5, 0.75, 1]

Symbolic Math

Symbolic computation keeps expressions in exact algebraic form rather than evaluating them to decimals. Calcario uses Algebrite as its CAS (Computer Algebra System).

Declaring Symbolic Variables

Before using a symbol in an expression, declare it with syms:

>> syms x y t

Once declared, any expression containing x, y, or t is automatically handled symbolically.

Differentiation

Compute derivatives using diff(expression, variable). Add a third argument for higher-order derivatives.

>> syms x
>> diff(x^4 - 3*x^2 + 2, x)
ans = 4*x^3-6*x

>> diff(sin(x)*cos(x), x)     % product rule applied
ans = cos(x)^2-sin(x)^2

>> diff(x^5, x, 2)             % second derivative d²/dx²
ans = 20*x^3

Integration

integrate(expression, variable) computes the indefinite integral (antiderivative). The constant of integration C is omitted.

>> integrate(x^2, x)
ans = 1/3*x^3

>> integrate(sin(x), x)
ans = -cos(x)

>> integrate(1/x, x)
ans = log(x)
For a definite integral ∫ₐᵇ f(x)dx, evaluate the antiderivative at the bounds manually or use: integrate(f, x, a, b) is not yet supported — compute as subs(F,b) - subs(F,a).

Factoring Polynomials

>> factor(x^2 - 1)
ans = (x-1)*(x+1)

>> factor(x^3 - x)
ans = x*(x-1)*(x+1)

Expanding Expressions

>> expand((x + 1)^3)
ans = x^3+3*x^2+3*x+1

>> expand((x + y)^2)
ans = x^2+2*x*y+y^2

Simplification

>> simplify(sin(x)^2 + cos(x)^2)
ans = 1

>> simplify((x^2 - 1)/(x - 1))
ans = 1+x

Limits

>> limit(sin(x)/x, x, 0)
ans = 1

>> limit((1 + 1/x)^x, x, inf)
ans = e

Taylor Series

taylor(expression, variable, order) — expands around 0 up to the given order:

>> taylor(sin(x), x, 5)
ans = x - 1/6*x^3 + 1/120*x^5

>> taylor(exp(x), x, 4)
ans = 1 + x + 1/2*x^2 + 1/6*x^3 + 1/24*x^4

Single-Variable Solve

Find roots of f(x) = 0:

>> solve(x^2 - 4, x)
ans = 2,-2

>> roots(x^2 - 5*x + 6, x)
ans = 2,3

Storing Symbolic Expressions

Assign symbolic results to variables for reuse:

>> syms x
>> f = x^3 - 3*x + 2
>> df = diff(f, x)
df = 3*x^2-3
>> factor(f)
ans = (x-1)^2*(x+2)

System of Equations Solver

Calcario can solve systems of linear equations in multiple unknowns. The solver uses Algebrite's symbolic differentiation to extract coefficients, then applies Math.js matrix factorization (LU decomposition) to find the solution.

Workflow

  1. Declare the unknowns as symbolic variables with syms.
  2. Define each equation as an expression equal to zero (the LHS).
  3. Call solve() with the equation names.
  4. Access individual results with dot notation.

Syntax

% Step 1 — declare unknowns
>> syms x y z

% Step 2 — define equations (LHS, set equal to zero)
>> eq1 = x + y - 5        % means x + y = 5
>> eq2 = 2*x - y - 1      % means 2x - y = 1

% Step 3 — solve
>> sol = solve(eq1, eq2)
sol:
  x = 2
  y = 3

% Step 4 — access results
>> sol.x
ans = 2
>> sol.y
ans = 3

Kirchhoff's Laws Example

Consider a circuit with three mesh currents I₁, I₂, I₃ governed by:

  • KCL node: I₁ + I₂ − I₃ = 0
  • KVL loop 1: 3I₁ − I₂ = 10
  • KVL loop 2: I₃ − 2I₁ = 50
>> syms I1 I2 I3
>> f1 = I1 + I2 - I3
>> f2 = 3*I1 - I2 - 10
>> f3 = I3 - 2*I1 - 50
>> solution = solve(f1, f2, f3)
solution:
  I1 = 30
  I2 = 80
  I3 = 110
>> solution.I1
ans = 30

How It Works Internally

For a system of n equations in n unknowns x₁…xₙ, Calcario:

  1. Extracts each coefficient aᵢⱼ = ∂fᵢ/∂xⱼ using symbolic differentiation.
  2. Evaluates each equation at x₁=…=xₙ=0 to find the constant term.
  3. Assembles the matrix system A·x = b.
  4. Solves using LU decomposition via Math.js.
Currently supports linear systems only. For nonlinear systems, rearrange algebraically into a linear form, or use roots() for single-variable polynomial equations.

Alternative Array Syntax

You can also pass equation lists explicitly with the array form:

>> solve([f1, f2, f3], [I1, I2, I3])
